# glyphforge

**glyphforge** is a lightweight Rust CLI tool that converts images and videos into stunning ASCII art — directly in your terminal.

> Turn pixels into characters, and media into monochrome masterpieces.

---

## Features

- Convert images (PNG, JPEG, BMP, etc.) piped via stdin into ASCII art
- Support for video streams when combined with tools like `ffmpeg`
- Fast, efficient, and minimal dependencies thanks to Rust
- Customizable output width and character sets (coming soon!)
- Perfect for terminal art, demos, or just showing off your media in text form

---

## Installation

Clone and run directly:

```bash
git clone https://github.com/yourusername/glyphforge.git
cd glyphforge
cargo run -- <options>
```

---

## Usage

- ffmpeg can be used in all cases

### Image from file

```bash
ffmpeg -i some_image.jpg -f image2pipe -vcodec ppm -s 1920x1080 - | glyphforge
```

### Video stream (using ffmpeg)

```bash
ffmpeg -i some_video.mp4 -vf scale=80:-1 -f image2pipe -vcodec ppm - | glyphforge
```

_Note:_ Adjust the scale filter to control output width.
